✈️ British Airways Review Dashboard

Interactive dashboard analyzing passenger reviews for British Airways using Tableau.
This project explores customer satisfaction trends across different service categories, aircraft types, and regions.

📊 Dashboard Preview

https://public.tableau.com/views/BritishAirwaysReviewDASHBOARD/Dashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

📁 Dataset

- ba_reviews.csv – passenger reviews and ratings

- Countries.csv – country mapping data for visualization

📈 Dashboard Features

- The dashboard allows interactive analysis through:

- KPI Summary – Average ratings across service categories

- Monthly Rating Trends – Changes in satisfaction over time

- Country Map – Average ratings by country

- Aircraft Comparison – Ratings by aircraft model

🔎 Business Insights

From the dashboard analysis:

- Cabin Staff Service receives the highest ratings, suggesting strong customer experience during flights.

- Entertainment has the lowest rating, indicating a potential area for service improvement.

- Some aircraft models consistently receive higher ratings, suggesting aircraft experience influences satisfaction.

- Ratings fluctuate over time, which could reflect operational changes or external factors affecting service quality.

These insights can help airlines prioritize service improvements and customer experience strategies.
